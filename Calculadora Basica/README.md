# Proyecto 01: Calculadora Interactiva con Validación de Datos

## 📝 Descripción
Este es mi primer proyecto desarrollado en Python, donde apliqué conceptos fundamentales de programación para crear una herramienta funcional y robusta. El enfoque principal fue la **experiencia del usuario** y la **prevención de errores** en tiempo de ejecución.

## 🚀 Características Técnicas
* **Manejo de Excepciones (`try-except`):** Implementado para garantizar que el programa no se detenga si el usuario ingresa texto en lugar de números o si intenta realizar una división entre cero.
* **Control de Flujo:** Uso de bucles `while True` para mantener la sesión activa y la sentencia `match-case` para un menú de opciones organizado y moderno.
* **Funciones Modulares:** Cada operación (Suma, Resta, División, Multiplicación, Par/Impar y Tablas) está encapsulada en su propia función para facilitar el mantenimiento del código.
* **Lógica de Validación:** El sistema incluye una función de verificación con contraseña para simular un acceso restringido.

## 🛠️ Herramientas y Conceptos
* **Lenguaje:** Python 3.x
* **Estructuras:** Bucles (`for`, `while`), Condicionales (`if`, `else`, `match`), y Tipos de datos (`float`, `int`, `string`).
* **Documentación:** El código fuente contiene comentarios detallados explicando el "por qué" de cada decisión técnica.

## 💡 Aprendizaje Personal
Durante el desarrollo de este script, aprendí la importancia de anticipar los errores del usuario. Mi mayor reto fue estructurar el bucle de la división para que, en caso de error, el usuario pudiera reintentar el ingreso del segundo número sin perder el progreso del primero.
